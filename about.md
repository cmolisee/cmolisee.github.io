This is a minimalistic Jekyll theme designed for clarity and focus. It emphasizes typography, generous whitespace, and thoughtful design details.

## Features

- Clean, refined aesthetic
- Responsive design
- Optimized for readability
- GitHub Codespaces ready
- SEO optimized with jekyll-seo-tag
- RSS feed support

## Getting Started

1. Open this repository in GitHub Codespaces
2. Wait for the container to build and Jekyll to start
3. Begin writing in the `_posts` directory
4. Customize `_config.yml` with your information

Enjoy writing.
